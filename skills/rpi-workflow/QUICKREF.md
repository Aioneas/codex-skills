## 核心原则

### 1. 上下文专注
- 每阶段清空上下文
- 有效上下文 ≤ 80K tokens
- 一次只做一件事

### 2. 约束驱动
- 约束 > 信息
- 告诉 AI "不要做什么"
- 缩小解决方案空间

### 3. 零决策执行
- Plan 阶段消除所有决策点
- Implementation 纯机械执行
- 无临场判断

### 4. 多模型协作
- 不同模型不同优势
- 交叉审查发现盲点
- 原型仅供参考，必须重写

## 约束集模板

### 硬约束（技术限制）
- [ ] 必须使用 X 技术
- [ ] 不能违反 Y 模式
- [ ] 必须满足 Z 性能要求

### 软约束（约定偏好）
- [ ] 遵循 X 风格指南
- [ ] 优先考虑 Y
- [ ] 避免 Z（除非必要）

### 依赖关系
- [ ] A 依赖 B
- [ ] C 必须在 D 之前完成

## PBT 属性类别

### 1. 交换律/结合律
```
f(a, b) == f(b, a)
f(f(a, b), c) == f(a, f(b, c))
```

### 2. 幂等性
```
f(f(x)) == f(x)
```

### 3. 往返一致性
```
decode(encode(x)) == x
```

### 4. 不变量保持
```
sum(before) == sum(after)
```

### 5. 单调性
```
timestamp[i] < timestamp[i+1]
```

### 6. 边界约束
```
0 <= value <= 100
len(result) <= max_size
```

## 常用提示词

### Research 阶段

**代码库探索**：
```
分析以下代码库，识别：
1. 现有结构和模式
2. 技术约束（不可违反的限制）
3. 风格约定（应遵循的规范）
4. 潜在风险
```

**约束提取**：
```
从以下需求中提取约束集：
- 硬约束：技术限制
- 软约束：约定偏好
- 依赖关系：实现顺序
```

### Plan 阶段

**模糊性检测**：
```
检测以下提案中的模糊性，列出：
[模糊性] <描述> → [需要的约束] <必须决定什么>
```

**隐含假设检测**：
```
识别以下提案中的隐含假设，指定：
[假设] <描述> → [需要的显式约束] <具体规范>
```

**PBT 属性提取**：
```
从以下需求中提取 PBT 属性：
[不变量] <数学属性> → [证伪策略] <如何生成反例>
```

### Implementation 阶段

**原型生成**：
```
生成以下任务的代码原型（unified diff patch 格式）：
<任务描述>

要求：
- 仅返回 diff patch
- 不要直接修改文件
```

**代码审查**：
```
审查以下代码变更，检查：
1. 逻辑正确性
2. 边界条件
3. 性能问题
4. 安全漏洞
5. 可读性和可维护性
```

## 故障排除

### 问题：上下文超限
**症状**：AI 响应变慢或出错  
**解决**：立即清空上下文，保存当前进度

### 问题：多模型调用失败
**症状**：`minis-model-use` 报错  
**解决**：检查模型配置和 API keys

### 问题：任务流无法执行
**症状**：Implementation 阶段遇到决策点  
**解决**：回到 Plan 阶段，补充缺失的约束

### 问题：代码审查意见冲突
**症状**：两个模型给出相反建议  
**解决**：优先采纳更保守的建议，或向用户说明

## 快速示例

### 最小化示例（单文件脚本）

```bash
# 1. 初始化
/var/minis/skills/rpi-workflow/init.sh /tmp/demo

# 2. Research（手动创建提案）
cat > /tmp/demo/.rpi/proposals/demo.md <<EOF
# 提案：Hello World 脚本

## 用户需求
创建一个 Python 脚本，打印 "Hello, World!"

## 约束集
### 硬约束
- 必须使用 Python 3
- 必须可执行

## 成功判据
- 运行脚本输出 "Hello, World!"
EOF

# 3. Plan（手动创建计划）
cat > /tmp/demo/.rpi/plans/demo-plan.md <<EOF
# 执行计划：Hello World 脚本

## 任务流
1. 创建 hello.py 文件
2. 添加 shebang 和代码
3. 设置可执行权限
4. 测试运行
EOF

# 4. Implementation（执行）
cd /tmp/demo
echo '#!/usr/bin/env python3' > hello.py
echo 'print("Hello, World!")' >> hello.py
chmod +x hello.py
./hello.py

# 输出：Hello, World!
```
